# Contributing

## Как вносить изменения
1. Сделайте fork репозитория.
2. Создайте ветку: `feature/short-name`.
3. Внесите изменения и проверьте работу сайта локально.
4. Откройте Pull Request в `main`.

## Рекомендации
- Не меняйте формат данных в `data/ege_2026.json` без обновления `app.js`.
- Для UI придерживайтесь существующей структуры и классов.
- Пишите понятные названия коммитов.
